# Sokoban
Play sokoban on your machine!

![sokoban-map](sokoban.png)

Tip: Hold buttons of movement or Undo actions for faster iterations.

With a built-in map editor, you can create your own levels and save them for later use.

![sokoban-collection](sokoban-collection.png)

![sokoban-map-editor](sokoban-map-editor.png)

## Download
Currently, only Linux is supported. Check releases to download the AppImage.
